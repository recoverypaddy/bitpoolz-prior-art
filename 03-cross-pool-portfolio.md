# DISCLOSURE 3: Cross-Pool Position Portfolio Management System

## Technical Field
Unified portfolio management for positions held across multiple simultaneous prize pools of different tiers and types.

## Background
Users entering multiple pools simultaneously have no unified view of their aggregate risk, exposure, or expected value across their entire position portfolio.

## Detailed Description
A portfolio management system aggregating positions held across multiple simultaneous prize pools into a unified dashboard. The system calculates and displays: total sats at stake across all pools, weighted average win probability across portfolio (weighted by sats per pool), aggregate expected value calculation (sum of: position_sats × win_probability × prize_amount for each pool), portfolio diversification score (concentration metric — percentage in highest single pool), time-to-draw timeline view showing all draw windows chronologically, and one-tap portfolio liquidation (list all positions for sale simultaneously at market-implied price). The system also provides portfolio rebalancing suggestions based on fill velocities: "Pool A is filling fast — your odds are now X. Pool B has slowed — consider increasing position." Historical portfolio performance tracking: win rate, average ROI on traded positions, total lifetime winnings.

## Key Claims
- Aggregate portfolio statistics across multiple pools
- Weighted average probability calculation
- Expected value aggregation formula
- Diversification concentration score
- One-tap multi-pool position liquidation
- Portfolio rebalancing suggestion engine
- Historical performance tracking across pools
